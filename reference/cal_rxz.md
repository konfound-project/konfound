# Calculate R2xz based on variances and standard error

Calculate R2xz based on variances and standard error

## Usage

``` r
cal_rxz(var_x, var_y, R2, df, std_err)
```

## Arguments

- var_x:

  variance of X

- var_y:

  variance of Y

- R2:

  coefficient of determination

- df:

  degrees of freedom

- std_err:

  standard error

## Value

R2xz value
