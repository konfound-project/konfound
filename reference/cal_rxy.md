# Calculate rxy based on ryxGz, rxz, and ryz

Calculate rxy based on ryxGz, rxz, and ryz

## Usage

``` r
cal_rxy(ryxGz, rxz, ryz)
```

## Arguments

- ryxGz:

  correlation coefficient between Y and X given Z

- rxz:

  correlation coefficient between X and Z

- ryz:

  correlation coefficient between Y and Z

## Value

rxy value
